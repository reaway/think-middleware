# think-middleware

## 安装
```bash
composer require reaway/think-middleware
```

## 用法
```php
use Think\Component\Middleware\Facade\Middleware;

require __DIR__ . DIRECTORY_SEPARATOR . 'vendor' . DIRECTORY_SEPARATOR . 'autoload.php';

```
